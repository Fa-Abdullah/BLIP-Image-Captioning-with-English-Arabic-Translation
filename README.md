# BLIP Image Captioning with English-Arabic Translation

This project generates descriptive captions for uploaded images using the BLIP image captioning model and automatically translates the generated caption into Arabic.

## Features

- Image caption generation using BLIP
- Automatic English-to-Arabic translation
- Interactive Gradio interface
- Supports image upload from local device
- Powered by Hugging Face Transformers

## Tech Stack

- Python
- PyTorch
- Hugging Face Transformers
- BLIP
- Helsinki-NLP Translation Model
- Gradio

## Workflow

1. Upload an image.
2. Generate an English caption using BLIP.
3. Translate the caption into Arabic.
4. Display both captions in the Gradio interface.

## Future Improvements

- Multiple language translation
- Download generated captions
- Batch image captioning
- Deploy on Hugging Face Spaces
